# Quickstart: Adding a New Chapter

This guide explains how to add a new chapter to the book.

1.  **Create a new markdown file:**
    -   Inside the `book/docs/<module-short-name>/` directory, create a new markdown file (e.g., `4-new-chapter.md`).

2.  **Add content to the chapter:**
    -   Write the content for the chapter in Markdown. You can use Docusaurus features like MDX.

3.  **Update the sidebar:**
    -   Open `book/sidebars.ts`.
    -   If you are using autogenerated sidebars, the new chapter will be added automatically based on the file's location. If you are using manual sidebars, add the new chapter to the appropriate module.

    ```typescript
    // Example from sidebars.ts for manual sidebar
    const sidebars: SidebarsConfig = {
      tutorialSidebar: [
        'intro',
        {
          type: 'category',
          label: 'Module 1',
          items: [
            '1-ros2-module-1/1-ros-fundamentals',
            '1-ros2-module-1/2-python-agents',
            '1-ros2-module-1/3-humanoid-modeling',
            '1-ros2-module-1/4-new-chapter', // Add the new chapter here
          ],
        },
      ],
    };
    ```

4.  **Build and test:**
    -   Run `npm run start` in the `book` directory to start the local development server.
    -   Check that the new chapter appears in the sidebar and that the content is rendered correctly.
